# FinIntel — RAG Financial Intelligence Dashboard

FinIntel is a real-time financial intelligence dashboard that integrates Retrieval-Augmented Generation (RAG) with live market data to provide analytical insights similar to a Wall Street terminal.

## Features
- Real-time market data (crypto + stocks via APIs)
- Interactive financial dashboard UI
- RAG-based query system for financial analysis
- Live ticker updates and KPI tracking
- Chart visualizations (S&P 500, sector flows)
- Integrated knowledge sources (FRED, SEC, NewsAPI)

## Tech Stack
- HTML, CSS (Custom UI Design)
- JavaScript (Vanilla JS)
- Chart.js (Data Visualization)
- APIs: CoinGecko, Alpha Vantage, Binance WebSocket
- LLM Integration (Anthropic Claude API)

## Highlights
- Designed a Bloomberg-style financial interface
- Implemented real-time streaming + fallback mechanisms
- Built a structured RAG prompt system for financial Q&A
